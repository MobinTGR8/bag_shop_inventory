Place your Inter font files here to match the entries in `pubspec.yaml`:

- `assets/fonts/Inter-Regular.ttf`
- `assets/fonts/Inter-Medium.ttf` (weight: 500)
- `assets/fonts/Inter-SemiBold.ttf` (weight: 600)
- `assets/fonts/Inter-Bold.ttf` (weight: 700)

You must add the actual `.ttf` files (not a README) for Flutter to bundle the fonts. If you don't have these fonts yet, download them from the official Inter repository or substitute with other fonts and update `pubspec.yaml` accordingly.
